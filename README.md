# IMPORT — Página para importaciones

Descripción
-----------
IMPORT es una página estática orientada a servicios de importación. Esta versión contiene la estructura HTML principal (index.html), estilos en la carpeta css, scripts en la carpeta js y recursos gráficos en img/. Este README explica la estructura de archivos, cómo abrir la página localmente.

Cómo ver el proyecto
--------------------
- Opción rápida: abrir index.html en el navegador (doble clic o "abrir con").
- Opción servidor local (recomendado para pruebas de fetch/servicios):
  - Python 3: `python -m http.server 8000` y luego abrir http://localhost:8000
  - Node (si tienes http-server): `npx http-server` o `npx serve`
- Para pruebas responsive, abrir herramientas de desarrollador en el navegador (F12) y alternar vistas de dispositivo.
